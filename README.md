# ReArtFlow

## 项目简介
ReArtFlow 是一个 **跨语言代码结构可视化** 平台，能够将不同语言的代码依赖转换为统一的 JSON 图谱，再生成 Mermaid 文本与 AntV G6 交互式思维导图。项目支持中文字符显示，适用于技术非专家快速了解项目结构。

## 核心特性
- **多语言解析**：通过 LLM 分析源码，抽取函数、类、模块之间的依赖关系。
- **统一图谱**：将依赖统一为 `{ "nodes": [...], "edges": [...] }` JSON 格式。
- **可视化**：
  - **Mermaid**：生成可嵌入文档的 Mermaid 流程图文本。
  - **AntV G6**：前端交互式思维导图，支持缩放、平移、节点点击。
- **中文支持**：完整保留 Unicode 中文字符，前端使用 Noto Sans SC 等字体确保不乱码。
- **Docker 部署**：提供 `Dockerfile` 与 `docker-compose.yml`，一键启动后端、前端服务。

## 快速开始
```bash
# 克隆仓库
git clone https://github.com/cyx-dzy/ReArtFlow.git
cd ReArtFlow

# 启动服务（Docker）
docker compose up -d

# 本地运行后端
uvicorn backend.api.app:app --reload

# 前端启动（假设已安装 Node）
cd frontend
npm install
npm run dev
```
